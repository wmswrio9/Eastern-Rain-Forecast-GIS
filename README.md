# 🛰️ ระบบสารสนเทศภูมิศาสตร์พยากรณ์และวิเคราะห์ปริมาณฝนล่วงหน้า 9 วัน ภาคตะวันออก
### **Eastern Thailand 9-Day NWP Rain Forecasting & GIS Analytics Portal**

ระบบสารสนเทศเชิงพื้นที่ (Interactive GIS Web Dashboard) สำหรับตรวจสอบและวิเคราะห์ปริมาณน้ำฝนพยากรณ์ล่วงหน้า 9 วัน จากแบบจำลองความละเอียดสูง (Numerical Weather Prediction - NWP API) ของกรมอุตุนิยมวิทยา เพื่อสนับสนุนการติดตาม เฝ้าระวัง และเตรียมความพร้อมบริหารจัดการน้ำในพื้นที่รับผิดชอบของ **สำนักงานชลประทานที่ 9 (ภาคตะวันออก)** ครอบคลุม 67 อำเภอ ใน 8 จังหวัด

---

## ✨ คุณสมบัติเด่นของระบบ (Key Features)

* 🗺️ **แผนที่เชิงพื้นที่แบบมีปฏิสัมพันธ์ (Interactive GIS Mapping):**
  * **รายอำเภอ / รายจังหวัด (Choropleth Maps):** แสดงเฉดสีปริมาณฝนเฉลี่ยตามขอบเขตการปกครอง
  * **🌈 เส้นชั้นน้ำฝน (Smooth Isohyet Contour):** คำนวณการกระจายตัวของฝนด้วย IDW & Gaussian Filtering เต็มกรอบสี่เหลี่ยมพื้นที่
  * **📦 ตารางกริดฝน (Grid Cells 5x5 กม.) & จุดกริดพยากรณ์ (3,762 จุด):** แสดงรายละเอียดความหนาแน่นของฝนรายจุด
* 📊 **ตารางวิเคราะห์ฝนรายวันและฝนสะสม (Analytical Tables):**
  * ตารางปริมาณฝนรายวันล่วงหน้า 9 วัน แยกรายอำเภอและรายจังหวัด
  * ตารางปริมาณฝนสะสม 3 วัน / 5 วัน / 7 วัน และฝนสะสมรวมตลอดช่วงพยากรณ์ 9 วัน
* 📄 **รายงานผู้บริหาร (Executive Report):**
  * สรุปภาพรวมสถานการณ์น้ำฝน พร้อมกราฟเปรียบเทียบแนวโน้มและปุ่มแคปเจอร์หน้าจอ (`.PNG`) สำหรับนำเสนอผู้บริหารได้อย่างรวดเร็ว

---

## 🛠️ โครงสร้างและการทำงานระบบ (Architecture)

1. **`fetch_eastern_rain.py`**: ดึงข้อมูลพยากรณ์ฝนรายจุดและสถานีหลักจาก TMD NWP API บันทึกลงฐานข้อมูล SQLite (`eastern_rain.db`)
2. **`spatial_processor.py`**: ประมวลผลจุดกริด 3,762 จุด ด้วยวิธีการคำนวณระยะทางแบบผกผัน (Inverse Distance Weighting: IDW) เพื่อหาค่าเฉลี่ยเชิงพื้นที่รายอำเภอและจังหวัด
3. **`generate_dashboard.py`**: สร้างหน้าเว็บเพจแบบ Single-File Standalone HTML (`dashboard.html` และ `executive_report.html`) พร้อมฝังชั้นข้อมูล GeoJSON และ Leaflet GIS

---

## 🚀 วิธีการเปิดใช้งานระบบ (Quick Start)

สามารถเปิดใช้งานผ่านเว็บเบราว์เซอร์ได้ทันทีโดยไม่ต้องติดตั้งเซิร์ฟเวอร์เสริม:
* ดับเบิลคลิกไฟล์ **`dashboard.html`** เพื่อใช้งานแผนที่และตารางพยากรณ์ฝนหลัก
* ดับเบิลคลิกไฟล์ **`executive_report.html`** เพื่อดูรายงานฉบับผู้บริหาร

### การอัปเดตข้อมูลพยากรณ์รอบใหม่
รันคำสั่งผ่าน Command Prompt / PowerShell:
```powershell
python fetch_eastern_rain.py
```
*(หมายเหตุ: ต้องนำ API Token จากกรมอุตุนิยมวิทยามาใส่ในไฟล์ `Token_TMD.txt` ก่อนรันระบบดึงข้อมูล)*
